# Chat Stream Reference

This folder includes a captured `/chat` Server-Sent Events stream for frontend reference.

Raw captured output:

```text
docs/chat_stream_reference.sse
```

Use this file as the current full example output for frontend parsing. It includes tool-call chunks, tool input, tool result data, assistant text deltas, the final agent update, `finish`, and `[DONE]`.

It can be replaced when the `/chat` streamed output changes.
